1.4 Enable Data Science Pipelines

Back to the introduction

This section will prepare your workspace for Section $ Data Science Pipelines. You can do this now to save time later, but you can also skip and return to this section.

If you do not intend to complete the pipelines section of the workshop you can skip this step and move on to the next section, Create a Workbench

Create a Pipeline Server

  • Click Create pipeline server

  • Select the Pipeline Artifacts data connection.

  • Leave the database configuration as the default.

  • Click Configure.

Pipelines should now be enabled for your project and the Create pipeline server button should disappear.
